# Personal Portfolio Website

Website portofolio pribadi yang menampilkan profil, keterampilan, proyek, serta informasi kontak dalam satu platform digital terstruktur.

---

## Deskripsi

Aplikasi ini merupakan web portofolio berbasis frontend yang dirancang untuk merepresentasikan identitas profesional secara visual dan informatif. Sistem menampilkan data diri, daftar skill, pengalaman, serta showcase proyek dalam tampilan modern, responsif, dan terorganisir.

---

## Fitur Utama

- Halaman profil dan ringkasan identitas
- Daftar skill dan teknologi yang dikuasai
- Showcase proyek dengan preview
- Navigasi single page
- Tampilan responsif (mobile & desktop)
- Dark dan Light mode

---

## Teknologi yang Digunakan

- HTML5  
- CSS3  
- JavaScript  
- React JS  
- Tailwind CSS  

---

## Struktur Halaman

**Home**  
Menampilkan hero section, perkenalan singkat, dan highlight utama.

**About**  
Berisi profil lengkap, latar belakang, dan deskripsi diri.

**Skills**  
Daftar kemampuan teknis, tools, dan teknologi yang dikuasai.

**Projects**  
Galeri proyek beserta deskripsi dan preview.

**Contact**  
Informasi kontak dan media komunikasi.
